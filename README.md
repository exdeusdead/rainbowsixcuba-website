## Main changes
- Restores tactical/cinematic Rainbow Six CUBA identity.
- Replaces generic/empty hero with a new HQ / operational command visual.
- Adds new contextual images for each main module:
  - Community
  - Events
  - Competitive
  - Statistics
  - Coaches
  - Partners
  - Collaborators
  - Incoming / Roadmap
  - Privacy
- Adds Companion Extension pages inside the website ecosystem:
  - /companion
  - /companion.html
  - /companion/privacy_policy.html
  - /companion/support.html
- Keeps language persistence across pages using localStorage and ?lang=.
- Includes ES, EN, FR, DE, PT, IT, ZH, JA language packages.
- Keeps legal/apolitical disclaimer.
- Build tested with npm run build.
