Rainbow Six CUBA Companion

Companion Extension

La extensión forma parte del ecosistema de estadísticas de Rainbow Six CUBA. Su objetivo es facilitar herramientas comunitarias, mejorar la visualización de datos y preparar una integración más fuerte entre Discord, website, API y perfiles públicos.

Ver política de privacidad Volver a estadísticas

Propósito

Apoyar el proyecto de estadísticas comunitarias y preparar funciones visuales para jugadores, equipos y rankings.

Datos

La extensión debe operar bajo una política transparente, evitando recopilar información no relacionada con el propósito comunitario.

Futuro

La visión es conectar datos del Discord y la API con vistas web públicas, ordenadas y de alta definición.

Qué buscamos construir

  • Una experiencia visual mejor que los embeds tradicionales de Discord.
  • Perfiles y estadísticas accesibles desde el dominio oficial.
  • Un puente claro entre comunidad, website, API y herramientas externas.
  • Un sistema transparente, no invasivo y orientado al crecimiento competitivo.
